
=== Paperclip ({{ .Chart.AppVersion }}) ===

{{- if eq .Values.service.type "NodePort" }}

Access Paperclip UI:

  On minikube:
    minikube service {{ include "paperclip.fullname" . }} --url

  Or directly:
    export NODE_PORT=$(kubectl get svc {{ include "paperclip.fullname" . }} -o jsonpath='{.spec.ports[0].nodePort}')
    export NODE_IP=$(kubectl get nodes -o jsonpath='{.items[0].status.addresses[0].address}')
    echo "http://${NODE_IP}:${NODE_PORT}"

{{- else if eq .Values.service.type "ClusterIP" }}

Access Paperclip UI via port-forward:

  kubectl port-forward svc/{{ include "paperclip.fullname" . }} 3100:{{ .Values.service.port }}
  open http://localhost:3100

{{- end }}
